### 기본적인 express JS 앱 구동
index.js
```js
const express = require('express')
const app = express()
const port = 3000

app.get('/', (req, res) => {
  res.send('Hello World!')
})

app.listen(port, () => {
  console.log(`Example app listening at http://localhost:${port}`)
})
```
* `require('express')` : `express` 모델을 가져옴
* `3000`번 포트로 로컬 서버 구동

package.json
```
"scripts": {
    "start": "node index.js",
    "test": "echo \"Error: no test specified\" && exit 1"
  },
```

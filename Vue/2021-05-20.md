## Vue.js External Install

<br>


### Vue-CLI

Install
```
$ npm install -g @vue/cli
```
* `-g` : 전역(전체) 모드로 설치

Create Project
```
$ vue create vue-movie-app
```
![](img/2021-05-20_Vue01.png)

<br>

![](img/2021-05-20_Vue02.png)
* default 선택

Run npm
```
$ cd vue-movie-app
$ npm run serve
```
![](img/2021-05-20_Vue03.png)
* 로컬 환경으로 실행

![](img/2021-05-20_Vue04.png)

<br>

### Vue-UI
* Vue.js 프로젝트 매니저 실행
* 브라우저로 UI 가 제공됨

```
$ vue ui
```
